# Contributing to PRONTO

## Commit Messages

The commit message's header should follow the [angular commit message format](https://gist.github.com/brianclements/841ea7bffdb01346392c):

```
<type>: <short summary>
  │           │
  │           │
  │           └─⫸ Short summary of the commit. No period at the end
  │
  │
  └─⫸ Commit Type: [build|feat|fix|aml|refactor|doc|test]
```


## Branches

Create a separate branch for each module or feature following the following naming convection:

```
<type>/<name>
  │       │
  │       │
  │       └─⫸ Branch name: Short and descriptive name for the branch.
  │
  └─⫸ Branch Type: feat|base|release|fix|(...)
  ```
