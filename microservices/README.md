# Microservices

## Datamodel

The data is stored into a [cloud-based service](https://render.com/) following a simple [model](https://editor.ponyorm.com/user/LarsHadidi/pdp_database/designer).

## Services

The services may form a service-mesh and serve all the functions that are required by frontoffice and backoffice.
