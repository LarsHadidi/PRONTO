# Research & Development

A precise formulation of the PDP-Problem can be found on Stinson's Paper [Designing Progressive Dinner Parties](
https://doi.org/10.48550/arXiv.2001.05394). He employs a deterministic construction method by making use of orthogonal latin squares.

In order to minimize an objective function, a specific formulation is necessary. Depending on the problem's formulation, we can employ different solution methods.
